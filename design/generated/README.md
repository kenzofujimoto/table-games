# Generated visual sources

The files in this directory preserve the original project-bound concept outputs used to build the production assets.

- `terrain-atlas.webp`: a 3×2 top-down terrain atlas generated for Auren, with forest, clay hills, pasture, wheat, stone and desert.
- `brand-concept.webp`: an identity study for the Table Games hub, combining a top-down table, four players and a central connection spark.

Production terrain crops live in `public/textures`. The final deterministic vector identity lives in `public/brand`.
